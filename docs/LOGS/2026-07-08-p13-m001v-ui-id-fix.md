# 2026-07-08 - P13-M001V quick_asset_box UI ID fix

Contexte : le smoke test humain P13-M001V a echoue a l'ouverture de la commande Fusion avec `name 'QUICK_ASSET_BOX_ASSETS_INPUT_ID' is not defined`.

Cause racine : `BoardGameInsertGenerator.py` referencait `QUICK_ASSET_BOX_ASSETS_INPUT_ID` dans `commandCreated` et `execute`, mais la constante n'etait pas definie dans le bloc module-level des IDs de CommandInputs. `py_compile` ne detectait pas ce cas car la reference est evaluee seulement au runtime Fusion dans le handler.

Correction : definition de `QUICK_ASSET_BOX_ASSETS_INPUT_ID = "bgig_quick_asset_box_assets"` avec les autres IDs UI, test AST de regression pour verifier que tout `*_INPUT_ID` reference est defini au niveau module, et marqueurs P13 ajoutes au check d'addin installe.

Statut : add-in reinstalle, settings P13 prepares, gate humaine P13-M001V a rejouer dans Fusion.
