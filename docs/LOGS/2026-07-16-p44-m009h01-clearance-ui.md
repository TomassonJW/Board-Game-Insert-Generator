# P44-M009H01 - Jeux repliés et X/Y commun

Date : 2026-07-16

Le retour humain sur la palette 0.1.31 confirme la logique de jeux par rôle
mais refuse leur présentation permanente dans les lignes techniques. Il corrige
aussi la saisie attendue : un seul jeu horizontal X/Y, distinct du jeu vertical
Z.

Le hotfix 0.1.32 déplace les overrides dans des volets repliés pour les assets,
plateaux/livrets et conteneurs. Une saisie X/Y écrit x et y ensemble. Les
valeurs anisotropes historiques restent préservées et explicitement signalées
jusqu’à une nouvelle saisie commune.

Le schéma, le moteur, les formules, les valeurs par défaut, le solveur, la CAD
IR et la scène Fusion ne changent pas. La preuve suivante est humaine dans
Fusion ; print-validated: false.
